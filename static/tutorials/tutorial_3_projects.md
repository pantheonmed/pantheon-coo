# Multi-step projects

**Duration:** ~10 minutes

## Steps

1. Open **Projects** (API: `POST /projects`) with a high-level goal spanning multiple tasks.
2. Watch sub-tasks appear and complete in order or parallel where safe.
3. Review aggregated logs in the project view.

## Example commands

Use the projects API body, e.g. goal: `Research three competitors and write a comparison markdown report in the workspace`.

## Expected result

A `project_id`, linked tasks, and final artifacts saved under `workspace/`.
