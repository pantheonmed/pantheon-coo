# Templates library

**Duration:** ~6 minutes

## Steps

1. Open **Templates** in the dashboard (or `GET /templates`).
2. Pick a template close to your industry (medical, finance, etc.).
3. Fill variables and run — or copy the command text into the command box.

## Example commands

- Run template `gst_report` with your quarter parameters.
- Run `linkedin_post` with topic and tone variables.

## Expected result

A queued task using the template’s pre-written command with your substitutions.
