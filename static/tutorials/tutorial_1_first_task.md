# Your first COO task

**Duration:** ~5 minutes

## Steps

1. Open the dashboard and sign in.
2. In the **command** box, type: `Check my system health and save a report to the workspace`.
3. Click **Execute** and watch the task list update.
4. Select the task and open the **Logs** tab to see live output.

## Example commands

- `Check my system health and save a report`
- `List files in the workspace and summarize what you find`

## Expected result

A completed task with score in **Evaluation**, a short summary, and report files under your workspace when filesystem tools are allowed.
